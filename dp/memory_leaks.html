Профилирование (Profiling)

Утечка памяти (memory leak) - процесс неконтролируемого уменьшения объёма свободной оперативной памяти (RAM) компьютера, связанный с ошибками в работающих программах, вовремя не освобождающих ненужные уже участки памяти, или с ошибками системных служб контроля памяти.

/*3*/   pointer = new char[100];
/*4*/ }
