Среды разработки

IDE -

  1. IntelliJ IDEA от компании JetBrains -
  2. Eclipse (затмение) - свободная интегрированная среда разработки (IDE). Развивается и поддерживается Eclipse Foundation. Первоначально Eclipse разрабатывалась фирмой IBM как преемник среды разработки IBM VisualAge, в качестве корпоративного стандарта IDE для разработки на разных языках под платформы IBM. По сведениям IBM, проектирование и разработка стоили 40 миллионов долларов (затем исходный код был полностью открыт).
  3. NetBeans - свободная интегрированная среда разработки приложений (IDE) на языках программирования Java, Python, PHP, JavaScript, C, C++, Ada.... Проект NetBeans IDE поддерживается и спонсируется компанией Oracle, однако разработка NetBeans ведется независимым сообществом разработчиков-энтузиастов (NetBeans Community) и компанией NetBeans Org.
